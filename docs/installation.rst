============
Installation
============
 
.. toctree::
   :maxdepth: 1

Dependencies
============

- ``numpy``
- ``scipy``
- ``msmtools``
- ``cython``

Installation via conda
======================

We recommend usage of the Anaconda/Miniconda Python distribution. To install thermotools, run this shell command:

::

   conda install -c omnia thermotools

Installation via pip
====================

For users of other Python distributions, we provide source releases via PyPI. Install the latest release via

::

   pip install thermotools

or run

::

   pip install git+https://github.com/markovmodel/thermotools.git@devel

to install the latest development stage.
